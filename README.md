# ⚡ Harry Potter Chatbot

An interactive, web-based chatbot experience designed for fans of the Wizarding World. Step into Hogwarts and converse with your favorite characters through a clean, responsive chat interface.

## 🌐 Live Demo
Experience the magic here: [harrypotter-chatbot.vercel.app](https://harrypotter-chatbot.vercel.app)

---

## ✨ Features
* **Magical UI:** An immersive theme inspired by the Harry Potter universe.
* **Interactive Conversations:** Engaging chat flows tailored to simulate wizarding dialogues.
* **Fully Responsive:** Optimized for a seamless experience across desktop, tablet, and mobile devices.
* **Fast & Lightweight:** Built entirely with vanilla web technologies for instant loading and smooth performance.

---

## 🛠️ Tech Stack
* **HTML5:** Semantic structural markup.
* **CSS3:** Custom wizarding themes, animations, and responsive layout designs.
* **JavaScript (ES6):** Dynamic message rendering and chat logic interaction.
* **Hosting:** Deployed and optimized on Vercel.
